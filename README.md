# ZFNavigationController

#### 特性

- 自定义导航滑动返回手势
- 播放视频、动画、滑动返回不卡顿
- 滑动pop的样式（阴影、背景颜色渐变）

#### 要求

- iOS 7+
- Xcode 8+

#### 用法

```objc
// 某一控制器禁止滑动手势 
self.zf_interactivePopDisabled = YES;
// 某一控制器不显示导航 
self.zf_prefersNavigationBarHidden = YES;
```
